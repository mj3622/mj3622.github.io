### 💼 项目介绍 | Project Introduction

> 日麻何切る在线练习 — 分类题库、牌面渲染与牌效解析，提升中盘切牌判断。React + TypeScript + Vite，在线：[nani-kiru.minjer.top](https://nani-kiru.minjer.top/)

::github{repo="mj3622/nani-kiru"}

> 轻小说下载器 — 基于 Python + Streamlit 的本地 Web 工具，从轻小说文库查询、下载小说与插图，支持分卷与 EPUB 导出。

::github{repo="mj3622/Wenku8Downloader"}

> 轻小说追更提醒 — 监控轻小说机翻网收藏列表的更新，有新章节时通过邮件、飞书机器人等推送，支持定时任务自动化。

::github{repo="mj3622/NoveliaReminder"}

> 日语新闻爬取学习工具 — 定时抓取 Mainichi/Asahi/Yahoo 等日文新闻，AI 清洗正文、翻译与语法点解析，邮件推送并附带每日生活短文。

::github{repo="mj3622/SakuraDigest"}

> Emoprobe 情感分析系统 — 针对 B 站视频的评论爬取与情感分析，支持视频链接/自定义文本分析，提供可视化统计与数据筛选。（参与后端开发）

::github{repo="pxxxl/Emoprobe"}

# 17. 愿望、提议与全面肯定

本课重点掌握如何表达内心的欲望（想要某物/想做某事），以及如何礼貌地邀请他人。

### 1. 想要某物：名 が 欲しいです

用于表达说话人想要得到某个**物品**。

- **句型**：`名词 + が + 欲（ほ）しい です`
- **注意**：对象语通常用助词 **が** 表示。
  - _例_：私（わたし）は カメラ **が 欲（ほ）しい です**。（我想要相机。）
  - _例_：友達（ともだち）**が 欲（ほ）しい です**。（我想要朋友。）

### 2. 想做某事：名を 动たいです

用于表达说话人想要进行某个**动作**。

- **变形**：动词去掉 `ます` + **たい**。
- **句型**：`名词 + を + 动词（たい形）+ です`
  - _注意_：虽然对象语可以用 `が`，但本课教材示例中使用了 `を`。
  - _例_：ビール **を** 飲（の）み**たい です**。（想喝啤酒。）
  - _例_：映画（えいが）**を** 見（み）**たい です**。（想看电影。）

### 3. 礼貌的邀请：动ませんか

用于邀请听话人一起做某事，或者委婉地提议。相当于中文的“不……吗？”、“要不要……？”。

- **句型**：`动词（否定形） + か`
- **语感**：因为是询问对方的意愿，所以比直接说“……吧”更加客气、尊重对方。
  - _例_：一緒（いっしょ）に 京都（きょうと）へ 行（い）き**ませんか**。（要不要一起去京都？）

### 4. 积极的提议/回应：动ましょう

用于积极地提议做某事，或者答应对方的邀请。相当于中文的“……吧”、“咱们……吧”。

- **句型**：`动词（ます形） + しょう`
  - _例_：ちょっと 休（やす）み**ましょう**。（稍微休息一下吧。）
  - _例_：はい、行き**ましょう**。（好，走吧/去吧。）

### 5. 全面肯定：疑问词 + でも

表示“无论……都……”。在疑问词后面加上 **でも**，表示在这个范畴内的所有事物都一样。

- **句型**：`疑问词 + でも + 肯定句`
- **常见组合**：
  - **何（なん）でも**：什么都……（例：何でも 食（た）べます / 什么都吃）
  - **いつでも**：什么时候都/随时……
  - **どこでも**：哪里都……
  - **だれでも**：谁都……

### 6. 实用表达与词汇

- **ぜひ**：副词，意思是“务必”、“一定”。

  - _用法_：常与“愿望”或“请求”的句型搭配，加强语气。
  - _例_：**ぜひ** 北海道（ほっかいどう）へ 行（い）きたい です。（**一定要**去北海道。）
- **今年中（ことしじゅう）に**：今年之内。后缀 `中` 在这里读作 `じゅう`，表示整个期间。
- **初詣（はつもうで）**：新年第一次去神社或寺庙参拜。

---

**本课总结：**

- 想要东西说 **欲（ほ）しい**。
- 想做动作说 **～たい**。
- 客气邀请说 **～ませんか**。
- 积极提议说 **～ましょう**。
- 什么都/随时都用 **～でも**。

# 18. 状态变化与人为改变

本课重点掌握两个动词：**なります**（变成）和 **します**（弄成/使变成），以及如何给出建议。

### 1. 自然的变化：～なります

表示事物性质或状态发生了自然改变，相当于中文的“变得……”。

- **一类形容词**：去 `い` + **く なります**

  - _例_：天気（てんき）が **暖（あたた）かく なります**。（天气变暖和。）
- **二类形容词 / 名词**：直接 + **に なります**

  - _例_：元気（げんき）**に なります**。（变得精神/恢复健康。）
  - _例_：もうすぐ ３０歳（さい）**に なります**。（快要到 30 岁了。）

### 2. 人为的改变：～します

表示人有意地改变事物的性质或状态，相当于中文的“把……弄成……”或“使……”。

- **一类形容词**：去 `い` + **く します**

  - _例_：音（おと）を **大（おお）きく します**。（把声音调大。）
- **二类形容词 / 名词**：直接 + **に します**

  - _例_：部屋（へや）を **きれいに します**。（把房间打扫干净。）
  - _点餐场景_：在餐厅决定点什么时，常用 `～に します`（我要……/我选……）。
  - _例_：カレー **に します**。（我要咖喱。）

### 3. 建议与选择：～ほうが いいです

用于在两者中进行比较选择，或者向对方提出建议，认为某种做法更好。

- **句型**：`一类形 / 二类形 + ほうが いいです`
  - _含义_：……比较好。
  - _例_：早（はや）い **ほうが いいです**。（早点比较好。）

### 4. 自动词与他动词（初步）

本课开始接触日语动词的一大难点——自他动词。

- **自动词**：描述事物自然发生的变化，不带宾语（助词用 `が`）。
- **他动词**：描述人对事物施加的动作，带宾语（助词用 `を`）。
- _本课关联_：`なります` 类似于自动词用法（变化发生），`します` 类似于他动词用法（去改变）。

### 5. 词汇辨析与实用表达

- **声（こえ）** 与 **音（おと）**：
- **声**：指人或动物发出的声音（嗓音、叫声）。
- **音**：指物体发出的声音（机器声、风声）。
- **間（ま）もなく** 与 **もうすぐ**：都表示“即将”、“马上”。
- **間もなく**：更郑重、正式（如电车广播）。
- **もうすぐ**：更口语化。
- **さらに** 与 **もっと**：都表示“更加”。
- **さらに**：郑重，多用于书面或正式场合。
- **もっと**：口语常用。
- **似合（にあ）います**：动词，意思是“适合”、“般配”（多指衣服、发型等）。

  - _例_：その 服（ふく）、**似合います**ね。（那件衣服很适合你呢。）
- **割引（わりびき）**：折扣。

  - _注意_：日语的“几折”和中文相反。`３割引` 意思是“扣除 3 成”，即 **7 折**。

---

**本课总结：**

- 自然变，用 **く/に なります**。
- 人为改，用 **く/に します**。
- 提建议，用 **ほうが いいです**。
- 人发声是 **声**，物体响是 **音**。
- **３割引** 是打 7 折。

# 19. 义务、禁止与动词“ない形”

本课重点掌握动词的 **“ない形”**（否定形），并利用它来表达“请不要做”、“必须做”以及“不通过做也可以”等日常生活中非常重要的规则和义务。

### 1. 动词的“ない形”

这是动词的否定形态，是后续语法变化的基础。

- **含义**：表示动作不发生。
- **变形示例**：
  - 行（い）きます → **行（い）かない**
  - 食（た）べます → **食（た）べない**

### 2. 请求不要做：动ないで ください

用于礼貌地请求对方**不要**进行某个动作，相当于中文的“请不要……”。

- **句型**：`动词（ない形）+ で ください`
  - _例_：ここ で 写真（しゃしん）を 撮（と）ら**ないで ください**。（请不要在这里拍照。）
  - _例_：無理（むり）を し**ないで ください**。（请不要勉强自己。）

### 3. 义务与责任：动なければ なりません

表示某种客观的义务或规定，不得不做某事，相当于中文的“必须……”。

- **句型**：`动词（ない形去い）+ ければ なりません`

  - _例_：薬（くすり）を 飲（の）ま**なければ なりません**。（必须吃药。）
- **口语表达**：**～ないと いけません**。

  - 意义相同，但在口语中更常用，听起来稍微随意一些。

### 4. 许可不做：动なくても いいです

表示没有必要做某事，相当于中文的“不……也可以”、“不用……”。

- **句型**：`动词（ない形去い）+ くても いいです`
  - _例_：明日（あした） 来（こ）な**くても いいです**。（明天不用来也可以。）
  - _例_：靴（くつ）を 脱（ぬ）がな**くても いいです**。（不用脱鞋也可以。）

### 5. 强调主语：名 が 名 です

当我们要询问或回答“谁是那个动作的主体”或者“哪一个是……”时，主语用助词 **が** 提示。

- **场景**：

1. **回答“哪一个”**：

- _问_：どの 人（ひと）が 田中（たなか）さん ですか。（哪个人是田中先生？）
- _答_：あの 人（ひと）**が** 田中（たなか）さん です。（那个人是田中先生。）

2. **新信息**：当主语是对方未知的新信息时。

### 6. 实用表达与词汇

- **大丈夫（だいじょうぶ）です**：没关系/不要紧/没问题。

  - 用于回应别人的担心或询问。
- **残業（ざんぎょう）**：加班。
- **何度（なんど）も**：好几次、多次。
- **だいぶ**：副词，表示程度，“相当”、“很”。

  - _例_：**だいぶ** 上手（じょうず）に なりました。（变得相当不错了。）
- **初心者（しょしんしゃ）** / **上級者（じょうきゅうしゃ）**：初学者 / 高级水平者。

# 20. 动词基本形与能力表达

本课重点掌握动词的**基本形**（也叫辞书形），并学习如何表达“会做某事”、“兴趣是什么”以及“在做某事之前”。

### 1. 动词的基本形（辞书形）

这是动词在词典中出现的原始形式，也是动词非礼貌体（简体）的基础。

- **特点**：所有动词的基本形都以 **う** 段假名结尾。
- **变形示例**：
  - 書（か）きます → **書（か）く**
  - 食（た）べます → **食（た）べる**
  - 来（き）ます → **来（く）る**

### 2. 表达能力：动词基本形 + ことが できます

用于表示具有某种能力，或者在某种环境下“可以/允许”做某事。

- **句型**：`动词（基本形）+ こと が できます`
- **能力**：

  - _例_：スミスさんは ピアノを **弾（ひ）く こと が できます**。（史密斯先生会弹钢琴。）
- **允许/可能**：

  - _例_：ここ で 写真（しゃしん）を **撮（と）る こと が できます**。（在这里可以拍照。）
- **对比**：之前学过 `名词 + が できます`（如 `スキーが できます`），现在学会了用动词表达更复杂的动作。

### 3. 描述兴趣爱好：趣味は 动词基本形 + ことです

当想说“我的爱好是做……”时，需要把动词名词化。

- **句型**：`私（わたし）の 趣味（しゅみ）は 动词（基本形）+ こと です`
  - _例_：私（わたし）の 趣味（しゅみ）は 切手（きって）を **集（あつ）める こと です**。（我的爱好是收集邮票。）
- _注意_：不能直接说 `趣味は 集めます`。

### 4. 时间顺序：动词基本形 + 前に

表示“在做某动作之前”。

- **句型**：`动词（基本形）+ 前（まえ）に、～`

  - _例_：寝（ね）る **前（まえ）に**、薬（くすり）を 飲（の）みます。（睡觉之前吃药。）
- **接名词**：`名词 + の + 前に`

  - _例_：食事（しょくじ）の **前（まえ）に** 手（て）を 洗（あら）います。（饭前洗手。）

### 5. 不确定指代：疑问词 + か

在疑问词（谁、什么、哪里）后面加上助词 **か**，就变成了表示不确定概念的词（某人、某事、某地）。

- **常见组合**：

  - **何（なに）か**：某样东西/什么东西

    - _例_：冬休（ふゆやす）み、**何（なに）か** しましたか。（寒假里做了什么（特别的）吗？）
  - **どこか**：某地/哪里
  - **だれか**：某人/谁
  - **いつか**：某时/迟早

### 6. 确认与共鸣：～よね

接在句尾，读升调。

- **用法**：表示征求对方同意，或者确认对方也知道的事实。相当于“……吧？”、“是……的吧”。
  - _例_：今日（きょう）は いい 天気（てんき）だ**よね**。（今天天气真好啊（你说是吧）。）

### 7. 实用表达与词汇

- **みんなで**：大家一起。强调集体行动。

  - _例_：**みんなで** 餃子（ギョーザ）を 食（た）べました。（大家一起吃了饺子。）
- **趣味（しゅみ）** 与 **興味（きょうみ）**：
- **趣味**：指具体的爱好（如看书、钓鱼）。
- **興味**：指对某事感兴趣，常搭配 `～に 興味（きょうみ）が あります`。
- **ごちそうします**：请客/做东。

  - _场景_：说“我来买单”时使用。

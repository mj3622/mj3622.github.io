# 13. 数量、频率与移动的目的

本课重点掌握数量词在句子中的位置，如何表达动作的频率，以及最重要的语法点——“去/来做什么”（移动的目的）。

### 1. 数量词的位置：名 [数量] 动

在日语中，数量词通常直接放在动词前面，**不需要**加任何助词。

* **句型** ：`名词 + を + 数量 + 动词`
  * *例* ：りんごを **４つ（よっつ）** 買（か）いました。（买了4个苹果。）
  * *注意* ：数量词不要放在名词前面（不说“4个苹果”），而要说“苹果 4个”。

### 2. 频率的表达：名 [时间] に 名 [次数] 动

用来表示“在多长的时间段内，进行几次动作”。

* **句型** ：`时间段 + に + 次数 + 动词`
  * *例* ：１週間（いっしゅうかん）**に** ２回（にかい） 映画（えいが）を 見（み）ます。（一周看两次电影。）
  * *例* ：１か月（いっかげつ）**に** １回（いっかい） 旅行（りょこう）します。（一个月旅行一次。）

### 3. 移动的目的：名 [场所] へ 动 [ます形] に 行きます

这是本课最核心的句型，表示“去某地是为了做某事”。

* **动词的“ます形”** ：指去掉 `ます` 剩下的部分（词干）。
  * *例* ：食（た）べます → **食（た）べ**
  * *例* ：飲（の）みます → **飲（の）み**
* **句型** ：`场所 + へ + 动词(去掉ます) / 名词(动作性) + に + 行（い）きます / 来（き）ます`
  * *例* ：日本（にほん）へ 勉強（べんきょう）**に** 来（き）ました。（来日本是为了学习。）
  * *例* ：居酒屋（いざかや）へ **飲（の）みに** 行（い）きます。（去居酒屋喝酒。）

### 4. 动作持续的时间：名 [时间] 动

表示动作持续了多久。与频率不同，这里的时间词后面**不加**助词 `に`。

* **句型** ：`时间段 + 动词`
  * *例* ：３時間（さんじかん） 勉強（べんきょう）しました。（学了3个小时。）
  * *例* ：どのぐらい かかりますか。（要花多长时间？）

### 5. 总计与限度：名 [数量] で

表示“合计”、“一共”或者“在这个数量范围内”。

* **句型** ：`数量 + で`
  * *例* ：全部（ぜんぶ）**で** ５００円（えん）です。（一共500日元。）
  * *例* ：１時間（いちじかん）**で** 行（い）けます。（1个小时能去/1个小时内能到。）

### 6. 实用词汇与表达

* **くらい / ぐらい** ：接在数量词后面，表示大概的数量，“左右”、“大约”。
* **とりあえず** ：常用短语。意思是“暂且”、“先”、“总之”。
  * *场景* ：在居酒屋点单时常说 `とりあえず ビール`（先来杯啤酒）。
* **数的数法** ：日语中不同的物品有不同的量词（本、枚、個、杯等），需要专门记忆。

---

**本课总结：**

* 数量词直接加在动词前（苹果 4个 买了）。
* 频率用 `时间 に 次数`（一周 に 2次）。
* 去干什么用 `去掉ます + に 行きます`（去 喝 に 行きます）。
* 一共多少用 `で`（全部 で）。

# 14. 动作进行、许可与禁止

本课重点掌握动词 **“て形”** 的进阶用法，用来描述正在发生的动作，以及日常生活中关于规则的表达。

### 1. 正在进行的动作：动词て形 + います

这是表示“现在进行时”的标准句型，表示动作此刻正在发生。

* **句型** ：`动词（て形）+ います`
  * *例* ：今（いま） 電話（でんわ）を かけ **て います** 。（现在正在打电话。）
  * *例* ：雨（あめ）が 降（ふ）っ **て います** 。（正在下雨。）

### 2. 表示许可：动词て形 + も いいです

用于表达“做某事也可以”或“允许做某事”。常用于请求别人的许可。

* **句型** ：`动词（て形）+ も いいです`
  * *例* ：写真（しゃしん）を 撮（と）っ **ても いいです** 。（可以拍照。）
* **提问与回答** ：
  * *问* ：タバコを 吸（す）っ **ても いいですか** 。（可以吸烟吗？）
    * *肯定回答* ：はい、いいです。（是的，可以。）/  **もちろん です** 。（当然可以。）
    * *否定回答* ：いいえ、いけません。（不，不行。）

### 3. 表示禁止：动词て形 + は いけません

用于告知对方“不准做某事”或“严禁做某事”。语气较强，常用于规定或提醒。

* **句型** ：`动词（て形）+ は いけません`
  * *例* ：ここで タバコを 吸（す）っ **ては いけません** 。（这里禁止吸烟。）
  * *例* ：**無理（むり）を しては いけません**よ。（不能勉强自己哦/别太累着了。）

### 4. 附着点与目的地：名 [场所/物体] に 动词

在使用表示“进入”、“乘坐”、“附着”等含义的动词时，动作的目标（附着点）用助词 **に** 表示。

* **典型动词** ： **乗（の）ります** （乘坐）、 **入（はい）ります** （进入）、 **座（すわ）ります** （坐下）。
* **句型** ：`名词 + に + 动词`
  * *例* ：電車（でんしゃ）**に** 乗（の）ります。（坐电车。）
  * *例* ：ボート **に** 乗（の）ります。（坐小船。）
  * *例* ：お風呂（ふろ）**に** 入（はい）ります。（泡澡/洗澡。）

### 5. 实用表达与医疗场景

本课涉及了一些在医院看病或关心他人的常用语。

* **お大事（だいじ）に** ：请多保重。
  * *用法* ：对生病或受伤的人说，祝愿对方早日康复。通常在探病结束或离开医院时说。
* **薬（くすり）を 出（だ）します** ：开药。
  * *注意* ：医生开药用动词 `出します`（拿出/开出）。
* **ゆっくり** ：副词，意思是“慢慢地”、“舒适地”、“好好地”。
  * *例* ：**ゆっくり** 休（やす）んで ください。（请好好休息。）
* **もちろん** ：当然。表示强烈的肯定。

---

**本课总结：**

* 正在做用  **て います** 。
* 请求许可说  **ても いいですか** 。
* 禁止做某事说  **ては いけません** 。
* 坐车、进门、泡澡，对象后面加助词  **に** 。
* 对病人要说  **お大事に** 。

# 15. 动词“て形”与请求表达

本课的重点是掌握动词的  **“て形”** 。学会这个变形，你就能把两个动作连起来说，还能请别人做某事。

### 1. 动词的“て形”

这是动词的一种变化形式，主要用于连接句子。

* **变形规则** （根据动词分类不同而不同，需要专门记忆）：
  * *例* ：食（た）べます → **食（た）べて**
  * *例* ：飲（の）みます → **飲（の）んで**
  * *例* ：行（い）きます → **行（い）って**

### 2. 请求某人做某事：～て ください

这是最常用的请求句型，相当于中文的“请（做）……”。

* **句型** ：`动词(て形) + ください`
  * *例* ：ちょっと 待（ま）っ **て ください** 。（请稍等一下。）
  * *例* ：この 本（ほん）を 読（よ）ん **で ください** 。（请读这本书。）

### 3. 动作的相继发生：～て 与 ～てから

当我们要按时间顺序描述两个以上的连续动作时，使用这两个句型。

* **简单的连接** ：`动词A(て形) + 动词B`
  * 表示动作A发生后，紧接着发生动作B。
  * *例* ：電話（でんわ）を し **て** 、友達（ともだち）に 会（あ）います。（打完电话去见朋友。）
* **强调顺序** ：`动词A(て形) + から + 动词B`
  * 表示“做完A **之后** ，（再）做B”。强调A动作结束是B动作开始的前提。
  * *例* ：手（て）を 洗（あら）っ **てから** 、ご飯（はん）を 食（た）べます。（洗完手**之后**再吃饭。）

### 4. 移动经过的场所：名 [场所] を 动

之前学过 `を` 提示宾语，这里 `を` 用于提示**移动经过的空间**或 **离开的场所** 。

* **经过** ：散步、跑步、飞过等。
  * *例* ：公園（こうえん）**を** 散歩（さんぽ）します。（在公园散步。）
* **离开** ：出去、下车等。
  * *例* ：家（うち）**を** 出（で）ます。（离开家/出门。）
  * *例* ：バス **を** 降（お）ります。（下公车。）

### 5. 更委婉的请求：～て くださいませんか

比 `～て ください` 更加礼貌、客气的请求方式，带有商量的语气。

* **句型** ：`动词(て形) + くださいませんか`
  * *含义* ：能请您……吗？ / 不知能否请您……？
  * *例* ：手伝（てつだ）っ **て くださいませんか** 。（能请您帮个忙吗？）

### 6. 实用表达与词汇

* **すみませんが** ：用于搭话或请求时的开场白，意思是“不好意思，……”、“劳驾，……”。
  * *作用* ：作为缓冲，让后面的请求听起来不那么突兀。
* **なかなか** ：副词，表示程度。
  * 常与否定形式连用，表示“怎么也（不）……”、“不容易……”。
  * *例* ：バスが **なかなか** 来（き）ません。（公车怎么也不来。）
* **買（か）い物（もの）します** ：购物。这是一个三类动词（不规则动词）。
* **お金（かね）を 下（お）ろします** ：取钱/提款。

---

**本课总结：**

* 请求别人用  **て ください** 。
* 更客气点用  **て くださいませんか** 。
* 先做A后做B，用 **て** 或者  **てから** 。
* 在公园散步、离开家，场所后面要用助词  **を** 。

# 16. 形容词连接与状态描述

本课重点掌握如何用一个句子描述事物的多个特征（既……又……），以及如何表达“某种状态持续中”。

### 1. 形容词与名词的并列连接

当我们要在一个句子中并列描述两个以上的性质（例如“既便宜又好吃”、“既安静又干净”）时，需要根据词性进行变形连接。

* **一类形容词（去“い”加“くて”）**
  * **规则** ：将词尾的 **い** 去掉，加上  **くて** 。
  * **句型** ：`一类形（去掉い）+ くて + 一类形`
    * **例** ：この 店（みせ）は 安（やす） **くて** 、おいしい です。（这家店既便宜又好吃。）
    * **例** ：背（せ）が 高（たか） **くて** 、脚（あし）が 長（なが）い です。（个子高，腿又长。）
* **二类形容词（加“で”）**
  * **规则** ：词干直接加  **で** 。
  * **句型** ：`二类形 + で + 二类形`
    * **例** ：この 公園（こうえん）は 静（しず）か  **で** 、広（ひろ）い です。（这个公园既安静又宽敞。）
* **名词（加“で”）**
  * **规则** ：名词直接加  **で** 。
  * **句型** ：`名词1 + で + 名词2`
    * **例** ：彼（かれ）は 日本人（にほんじん） **で** 、留学生（りゅうがくせい）です。（他既是日本人，又是留学生。）

### 2. 动作结果的状态：动词て形 + います

之前学过 `～て います` 表示“正在进行”，本课学习它的另一种用法：表示 **动作结果的状态持续** 。

* **用法** ：常用于瞬间动词（如：结婚、拿着、住、知道、拥有）。
* **句型** ：`动词（て形）+ います`
  * **例** ：結婚（けっこん）し **て います** 。（已婚/结婚了。）
  * **例** ：東京（とうきょう）に 住（す）ん **て います** 。（住在东京。）
  * **例** ：カメラを 持（も）っ **て います** 。（有/拿着相机。）
  * **例** ：IDカードは  **ありますか** 。（注意：问“有没有带在身上”时也常用 `持っていますか`，但这里例句用了存在的 `あります` 确认是否持有）

### 3. 逆接（转折）：～が

用于连接两个句子，表示转折关系，相当于中文的“虽然……但是……”。

* **句型** ：`句子1 + が、句子2`
  * **例** ：日本（にほん）の 食（た）べ物（もの）は おいしい です **が** 、高（たか）い です。（日本的食物虽然好吃，但是很贵。）

### 4. 尚未发生：まだ + 否定

表示某动作目前还没有发生或完成。

* **句型** ：`まだ + 动词（否定形）`
  * **例** ：**まだ** 食（た）べ **ません** 。（还没吃。）
* **对比** ：`もう` 是“已经”，`まだ` 是“还未”。

### 5. 实用表达与词汇

* **ちゃんと** ：副词，表示“规规矩矩地”、“好好地”、“准确无误地”。
  * **例** ：**ちゃんと** 勉強（べんきょう）します。（好好学习。）
* **ずいぶん** ：副词，表示程度很高，“相当”、“非常”。含有惊讶的语气。
* **ほら** ：感叹词，用于引起对方注意，相当于“你看！”、“喏！”。
* **派手（はで）** ：二类形容词，意思是“花哨”、“艳丽”、“排场大”。反义词是“地味（じみ）”。
* **嫌（きら）いじゃ ありません** ：委婉表达。直译是“不是讨厌”，实际意思是“还算喜欢”或“不排斥”。

---

**本课总结：**

* 形容词连接：一类形去 **い** 加  **くて** ，二类形直接加  **で** 。
* 名词连接也用  **で** 。
* 住在哪里、结婚了，都用 **て います** 表示状态。
* “虽然……但是……”用助词  **が** 。
* “还没做”用 **まだ** 加否定。

# 5. 时间表达与动词变位

本课的重点是从静态的“描述事物”转向动态的“描述动作”，学会表达“现在几点”以及“在什么时间做什么事”。

### 1. 时间的表达：今 ～時 ～分 です

这是询问和回答当前时间的最基本句型。

* **句型** ：`今（いま） ～時（じ） ～分（ふん/ぷん） です`。
* **注意** ：“分”的读音会根据前面的数字发生变化（如：五分读  **ごふん** ，十分读  **じゅっぷん** ），需要专门记忆。

### 2. 动词的“ます”形变位

日语动词通过词尾变化来表示时态和肯定/否定。本课学习的是礼貌体（**ます**形）的四种基础形态。

* **非过去（现在/将来）** ：
  * 肯定：**～ます** （例：起き**ます** / 起床）
  * 否定：**～ません** （例：起き**ません** / 不起床）
* **过去** ：
  * 肯定：**～ました** （例：起き**ました** / 起床了）
  * 否定：**～ませんでした** （例：起き**ませんでした** / 没起床）

### 3. 动作发生的时间点：名 [时间] に 动词

想要说明动作发生在具体几点几分时，时间名词后面必须加助词  **に** 。

* **句型** ：`时间 + に + 动词`
  * *例* ：毎朝（まいあさ） ６時（ろくじ） **に** 起（お）きます。（每天早上6点起床。）
  * *注意* ：如果时间词本身不含数字（如“今”、“昨日”、“今日”），通常**不用**加 `に`。

### 4. 时间段：～から ～まで

表示动作开始和结束的时间范围，相当于中文的“从……到……”。

* **句型** ：`时间A から 时间B まで 动词`
  * *例* ：９時（くじ） **から** ５時（ごじ） **まで** 働（はたら）きます。（从9点工作到5点。）
* **用法** ：`から`（从）和 `まで`（到）也可以单独使用。

### 5. 询问时间：いつ

除了问“几点”（何時），还可以用  **いつ** （什么时候）来询问时间。

* **特点** ：`いつ` 后面**不能**加助词 `に`。
  * *例* ：**いつ** 日本（にほん）へ 行（い）きますか。（什么时候去日本？）

### 6. 左右模糊的时间：～ごろ

表示大概的时间点，相当于中文的“……左右”。

* **用法** ：接在时间词后面。
  * *例* ：６時（ろくじ） **ごろ** 起（お）きます。（6点**左右**起床。）
  * *对比* ：`に` 表示精确时间点，`ごろ` 表示大约时间点，`ごろ` 后面通常不加 `に`。

### 7. 助词“は”的对比用法

助词 **は** 除了提示主题外，还可以表示 **对比** 。

* **场景** ：当你想要强调“这个时间不做，那个时间做”或者“我不做，他做”时，用 `は` 来加强语气。
  * *例* ：昨日（きのう）**は** 勉強（べんきょう）しませんでした。（强调“昨天”没学，可能今天学了。）

### 8. 职场称呼礼仪

在工作单位称呼他人时，需要注意身份。

* **称呼上级** ：通常使用“姓氏 + 职位”或者直接用“职位名”。
  * *例* ：田中（たなか）部長（ぶちょう）（田中部长）。
* **称呼自己** ：在介绍自己时，千万不能加“さん”或职位敬称。

---

**本课总结：**

* 动词分四态： **ます / ません / ました / ませんでした** 。
* 具体时间点用助词  **に** 。
* 时间范围用 **から** (从) 和 **まで** (到)。
* 问“什么时候”用 **いつ** (不加に)。
* 大概时间用  **ごろ** 。

# 6. 移动与交通

本课的核心动词通常是  **行（い）きます** （去）、 **来（き）ます** （来）、 **帰（かえ）ります** （回），围绕这些动作，我们学习如何添加各种信息。

### 1. 目的地：名 [场所] へ 动词

表示移动的方向或目的地时，使用助词  **へ** 。

* **注意** ：这里的 **へ** 读作 **え**
* **句型** ：`场所 + へ + 行きます / 来ます / 帰ります`
  * *例* ：学校（がっこう）**へ** 行（い）きます。（去学校。）

### 2. 交通手段：名 [交通工具] で 动词

表示使用某种交通工具进行移动时，使用助词  **で** 。

* **句型** ：`交通工具 + で + 移动动词`
  * *例* ：電車（でんしゃ）**で** 行（い）きます。（坐电车去。）
  * *例* ：自転車（じてんしゃ）**で** 帰（かえ）ります。（骑自行车回去。）
* **例外** ：如果是“走路”去，不说“散步で”，而说  **歩（ある）いて** 。
  * *例* ：歩（ある）いて 帰（かえ）ります。（走着回去。）

### 3. 同行者：名 [人] と 动词

表示“和某人一起”做动作时，使用助词  **と** 。

* **句型** ：`人 + と + 动词`
  * *例* ：友達（ともだち）**と** 来（き）ました。（和朋友一起来了。）
* **例外** ：如果是“独自一人”，不说“一人と”，而说  **一人（ひとり）で** 。

### 4. 移动的起点与终点：から ～ まで

上一课学过 `から/まで` 用于时间，它们同样可以用于空间场所，表示“从某地到某地”。

* **句型** ：`起点 + から + 终点 + まで + 移动动词`
  * *例* ：家（うち）**から** 学校（がっこう） **まで** 。（从家到学校。）
  * *注意* ：也可以只用其中一个，比如 `会社（かいしゃ）から`（从公司出发）。

### 5. 助词的叠加：助词 + は

为了表示**对比**或 **强调** ，助词 `は` 可以叠加在其他助词后面。

* **常见组合** ：`には`、`では`、`へは`、`からは`、`までは`、`とは`。
  * *例* ：日曜日（にちようび）**には** 行（い）きません。（强调“周日”这天不去，可能别的日子去。）

### 6. 实用表达与词汇辨析

本课包含了一些非常地道的口语表达：

* **お疲（つか）れ様（さま）でした** ：辛苦了。这是职场和日常生活中最常用的寒暄语，用于工作结束或分别时。
* **たしか** ：表示记忆模糊，相当于中文的“好像是……”、“如果我没记错的话……”。
  * *例* ：**たしか** 金曜日（きんようび）です。（（如果不记错的话）好像是周五。）
* **家 vs うち** ：两者都指“家”。
* **家（いえ）** ：多指物理上的房子、建筑物。
* **うち** ：多指心理上的归属，“我家”、“我的家”，更口语化。
* **まっすぐ** ：笔直地、直接地。
  * *例* ：**まっすぐ** 帰（かえ）りました。（径直回家了，没去别的地方。）

---

**本课总结：**

* 往哪走，用 **へ**。
* 怎么去，用 **で** (工具)。
* 和谁去，用 **と** (人)。
* 从哪到哪，用 **から** 和  **まで** 。
* 下班别忘说  **お疲（つか）れ様（さま）でした** 。

# 7. 动作对象与购物请求

本课的重点在于如何详细描述“做什么”（对象）、“在哪里做”（场所），以及在商店买东西时如何表达“请给我这个”。

### 1. 动作的对象：名 を 动词

这是日语中最常见的句型之一，用于表示动作直接作用的对象。

* **助词用法** ：**を** 用于标记宾语（动作的对象）。
  * *注意* ：虽然作为假名它属于“wa”行，但在作助词时发音同“o”。
* **句型** ：`名词 + を + 动词`
  * *例* ：ご飯（はん）**を** 食（た）べます。（吃饭。）
  * *例* ：テレビ **を** 見（み）ます。（看电视。）

### 2. 动作的场所：名 [场所] で 动词

在第六课我们学过 `で` 表示交通工具，在本课中，**で** 表示动作进行的 **场所** 。

* **句型** ：`场所 + で + 动词`
  * *例* ：食堂（しょくどう）**で** ご飯（はん）を 食（た）べます。（在食堂吃饭。）
* **辨析** ：
  * `に`：表示存在的地点（在）。
  * `で`：表示动作发生的地点（在...做...）。

### 3. 选择：名 か 名

用于列举两个或以上的名词，表示“或者”。

* **句型** ：`名词A か 名词B`
  * *例* ：パン **か** 卵（たまご）。（面包或者鸡蛋。）

### 4. 购物请求：名 を ください

在商店买东西或餐厅点餐时的必备句型，意为“请给我……”。

* **句型** ：`名词 + を ください`
  * *例* ：これ  **を ください** 。（请给我这个。）
  * *例* ：水（みず） **を ください** 。（请给我水。）

### 5. “何”的读音规则：なん vs なに

汉字“何”在日语中有两个读音，需要根据后面的音节来判断。

* **读作“なん”的情况** ：

1. 后面接的词以 **t、d、n** 开头的音节时（如：ですか、だ、の）。
   * *例* ：**何**ですか（是什么）、**何**の（什么的）。
2. 后面接量词时。
   * *例* ：**何**歳（几岁）、**何**時（几点）。

* **读作“なに”的情况** ：
* 除了上述情况外，通常读作  **なに** 。
* 尤其是在助词 **を** 前面。
  * *例* ：**何**を 買（か）いますか。（买什么？）

### 6. 实用会话与表达

本课介绍了一些让对话更流畅的短语：

* **そうですか** ：是吗？（读降调）。表示理解了对方说的话，相当于“噢，原来是这样”。
* **そうですね** ：是啊。（读降调）。表示同意对方的看法，或者在思考回答时作为缓冲（相当于“嗯……让我想想”）。
* **じゃあ** ：那么……。用于承接上文，提出建议或转换话题。
* **すみません** ：不好意思。除了表示道歉，在商店招呼店员（相当于“劳驾”、“服务员”）时也用这个词。

---

**本课总结：**

* 吃什么、喝什么，用助词  **を** 。
* 在哪做，用助词  **で** 。
* 想要什么东西，说  **～を ください** 。
* 问“干什么”时，“何”要读作  **なに** 。

# 8. 手段工具与授受关系

本课的重点在于如何表达“用什么做某事”，以及如何描述“我给别人东西”或“别人给我东西”。

### 1. 手段与工具：名 [工具] で 动词

助词 **で** 在这里表示手段、原材料或工具。

* **句型** ：`工具/手段 + で + 动词`
  * *例* ：箸（はし）**で** 食（た）べます。（用筷子吃。）
  * *例* ：日本語（にほんご）**で** 手紙（てがみ）を 書（か）きます。（用日语写信。）

### 2. “给”与“受”：あげます / もらいます

这是日语中最基础的授受动词，必须分清方向。

* **给别人 (Give)** ：**あげます**
* **句型** ：`送礼人 は 收礼人 に 物品 を あげます`
  * *例* ：私（わたし）は 田中（たなか）さん **に** 本（ほん）を  **あげます** 。（我给田中先生书。）
* **从别人那里得到 (Receive)** ：**もらいます**
* **句型** ：`收礼人 は 送礼人 に 物品 を もらいます`
  * *例* ：私（わたし）は 田中（たなか）さん **に** 花（はな）を  **もらいます** 。（我从田中先生那里收到了花。）

### 3. 助词“に”与“から”的转换

在表示“得到” (`もらいます`) 时，助词 **に** 表示来源。如果来源是“人”，可以用 **から** 替换  **に** ，意思完全一样。

* **句型** ：`人 + から + もらいます`
  * *注意* ：如果来源是“公司”或“学校”等组织机构，通常更倾向于使用  **から** 。
  * *例* ：会社（かいしゃ）**から** 時計（とけい）を もらいました。（从公司得到了一块表。）

### 4. 见面的对象：名 [人] に 会います

表示“和某人见面”时，见面的对象用助词 **に** 标记，动词是  **会（あ）います** 。

* **句型** ：`人 + に + 会います`
  * *例* ：友達（ともだち）**に** 会（あ）います。（见朋友。）
  * *注意* ：这里不能用助词 `を`。

### 5. 状态的完成：もう

副词 **もう** 表示“已经”。常用于确认某事是否已经完成。

* **句型** ：`もう + 动词(过去式)`
  * *问* ：**もう** 昼（ひる）ご飯（はん）を 食（た）べましたか。（已经吃过午饭了吗？）
  * *答* ：はい、**もう** 食（た）べました。（是的，已经吃过了。）

### 6. 语气助词：よ

接在句尾，表示 **提醒** 、**告知**听话人不知道的信息，或者加强语气。

* **用法** ：读升调或重读。
  * *例* ：この 映画（えいが）は 面白（おもしろ）い です **よ** 。（这部电影很有趣哦！——含有推荐的语气）

### 7. 实用词汇与表达

本课还涉及了一些日常沟通的高频词：

* **さっき / たった今** ：表示“刚才”、“刚刚”。`たった今` 比 `さっき` 离现在的时间更近。
* **分かりました** ：明白了/知道了。表示接受指令或理解对方的话。
* **お願いします** ：拜托了/麻烦了。用于请求对方做某事。

---

**本课总结：**

* 用什么工具，用助词  **で** 。
* 我给别人，用  **あげます** 。
* 别人给我（我收到），用  **もらいます** 。
* 见谁，用助词 **に** 加  **会います** 。
* “已经”做完了，用  **もう** 。

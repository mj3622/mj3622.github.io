# 1. 基础句型与自我介绍

本课将介绍如何用最简单的结构介绍自己、描述事物，以及区分相似的词汇。

### 1. 名词 1 は 名词 2 です

这是日语中最基本的**判断句**结构，相当于中文的“A 是 B”。

- **含义**：`は` 是助词（读作 wa），提示句子的主题；`です` 表示断定和礼貌。
- **例句**：
- 私（わたし）**は** クリス **です**。（我是 Chris。）
- 私 **は** 学生（がくせい）**です**。（我是学生。）

### 2. 名词 1 は 名词 2 ではありません

这是 `です` 的**否定形式**，相当于中文的“A 不是 B”。

- **含义**：`ではありません` 是标准礼貌否定，口语中常说成 `じゃありません`。
- **例句**：
- 私 **は** 会社員（かいしゃいん） **ではありません**。（我不是公司职员。）
- 彼（かれ） **は** 日本人（にほんじん） **ではありません**。（他不是日本人。）

### 3. 疑问句及应答 (…か)

日语的问句通常不使用问号，而是在句尾加上助词 `か`，读升调。

- **结构**：句子 + `か`？
- **应答**：肯定用 `はい`（是），否定用 `いいえ`（不）。
- **例句**：
  - **问**：あなた **は** 学者（がくしゃ） **ですか**。（你是学者吗？）
  - **答**：いいえ、学者 **ではありません**。（不，我不是学者。）

### 4. 名词 1 の 名词 2

助词 `の` 用于连接两个名词，表示**所属、属性**或**从属关系**，相当于中文的“的”。

- **用法**：

1. **所属机构**：CUHK **の** 学生（港中大的学生）。
2. **国家/产地**：日本 **の** カメラ（日本的相机）。
3. **所有物**：私 **の** 本（我的书）。

### 5. 人称与省略

在日语对话中，如果根据上下文能清楚知道主语是谁（特别是“我”和“你”），通常会**省略主语**，这样显得更自然。

- **现象**：不用每次都说 `私は...`。
- **例子**：
  - (你是) 学生吗？ -> 学生ですか。 (省略了“あなたは”)
  - (我是) 中国人。 -> 中国人です。 (省略了“私は”)

### 6. 寒暄语：どうぞ よろしく お願いします

这是自我介绍结束时的必用语，非常重要。

- **含义**：直译为“请多关照”，在初次见面时表示“很高兴认识你，接下来请多指教”。
- **用法**：说完自己的名字或身份后，紧接着说这句话。

### 7. “会社員” vs “社員”

这两个词都表示“公司职员”，但用法有细微区别，容易混淆。

- **会社員**：表示**职业**本身。

  - _例_：私 は **会社員** です。（我是公司职员。）
- **社員**：侧重于**归属**，通常前面要接公司名称（用到助词 `の`）。

  - _例_：私 は Google **の 社員** です。（我是 Google 的员工。）
  - _注意_：不能单说 `私は社員です`。

# 2. 指示代词与礼貌表达

本课重点讲解如何用日语指代事物（这个、那个、哪个），以及学习更地道的礼貌用语和寒暄表达。

### 1. 指示代词：これ / それ / あれ

这是用来指代**事物**的代词，相当于中文的“这个、那个”。

- **これ**（这个）：离说话人近的物体。
- **それ**（那个）：离听话人近的物体。
- **あれ**（那个）：离说话人和听话人都远的物体。
- **句型**：`これ / それ / あれ は 名词 です`
  - _例_：これは 本（ほん）です。（这是书。）

### 2. 提问：だれ / 何 / どれ

本课介绍了三种基础疑问词：

- **だれですか**（是谁？）：询问人。

  - _注意_：更礼貌的说法是 `どなたですか`。
- **何（なん）ですか**（是什么？）：询问事物。
- **どれ**（哪一个？）：用于在三个或以上的事物中进行选择。

  - _例_：あなたの 傘（かさ）は **どれ** ですか。（你的伞是哪一把？）

### 3. 指示连体词：この / その / あの

这组词也表示“这、那”，但**不能单独使用**，后面必须紧跟一个名词。

- **用法**：修饰名词，用来特指某个人或物。
- **句型**：`この / その / あの + 名词 は 名词 です`
  - _例_：**この** カメラ は 私のです。（**这台**相机是我的。）
- **区别**：`これ` 是代词（单独作主语），`この` 是连体词（必须粘着名词）。

### 4. 礼貌语言：方 (かた) 与 询问年龄

日语非常注重礼貌程度，本课引入了针对长辈或初次见面者的敬语表达。

- **方 (かた)**：是 `人 (ひと)` 的尊称。

  - _例_：あの **方** は どなた ですか。（那位是哪位？——比“那个人是谁”更礼貌）
- **询问年龄**：

  - 普通：何歳（なんさい）ですか。
  - 礼貌：**おいくつ** ですか。（您贵庚？）

### 5. 应答：はい 与 ええ

两者都表示肯定的“是”，但使用场景不同。

- **はい**：标准、正式。用于正式场合或点名应答。
- **ええ**：口语、随意。用于日常对话，相当于“嗯、对”。

### 6. 常用寒暄：どうぞ 与 谢谢

- **どうぞ**：在请别人先行、吃东西或递给别人东西时使用，意为“请”、“给”。
- **どうも ありがとうございます**：非常感谢。
- `どうも` 既可以加强感谢的语气，也可以在非正式场合单独使用，表示轻微的“谢了”。

---

**本课总结：**

- 离自己近用 `こ` 系列 (これ/この)
- 离对方近用 `そ` 系列 (それ/その)
- 离大家都远用 `あ` 系列 (あれ/あの)
- 疑问词用 `ど` 系列 (どれ/どの)

# 3. 场所位置与购物询价

本课的重点在于如何描述我们在哪里、东西在哪里，以及在商店购物时如何询问价格。

### 1. 指示场所：ここ / そこ / あそこ

与上一课的“事物”对应，这组词用于指代**场所**（这里、那里）。

- **含义**：

  - **ここ**（这里）：说话人所在的场所。
  - **そこ**（那里）：听话人所在的场所。
  - **あそこ**（那里）：离两人都远的场所。
- **句型**：`ここ / そこ / あそこ は 名词(场所) です`

  - _例_：**ここ** は 教室（きょうしつ）です。（这里是教室。）

### 2. 询问与表达位置：どこ

这是本课的核心句型，用于确认人或物体的所在地。

- **陈述句**：`名词(人/物) は 名词(场所) です`

  - _例_：トイレ は **あそこ** です。（洗手间在那边。）
- **疑问句**：`名词 は どこ ですか`（...在哪里？）

  - _例_：受付（うけつけ）は **どこ** ですか。（接待处在哪里？）

### 3. 助词“も”：表达“也”

当后一句的主题与前一句相同时，使用助词 `も` 来代替 `は`，相当于中文的“也”。

- **句型**：`名词A も 名词B です`
  - _例_：
    - 田中（たなか）さんは 学生（がくせい）です。（田中先生是学生。）
    - 私（わたし）**も** 学生（がくせい）です。（我**也**是学生。）

### 4. 选择疑问句：是 A 还是 B？

日语中的选择问句不需要专门的“还是”这个词，只需要把两个疑问句连起来即可。

- **句型**：`句子1 か，句子2 か`
  - _例_：ここは 3 階（さんがい）ですか、4 階（よんかい）ですか。（这里是 3 楼还是 4 楼？）
  - _注意_：回答时不需要说 `はい` 或 `いいえ`，直接回答正确的选项即可。

### 5. 询问价格：いくら

购物时的必备用语。

- **句型**：`名词 は いくら ですか`
  - _例_：この カメラ は **いくら** ですか。（这台相机多少钱？）

### 6. 实用表达与礼貌用语

本课还涉及了一些让对话更自然的细节：

- **1 階 (层数表达)**：楼层的数法中，“1 楼”读作 **いっかい**，是促音，需要特别注意。
- **あのう (搭话)**：在向陌生人问路或搭话时，先说一声 `あのう...`（那个...）可以表示客气和犹豫。
- **礼貌语言 (こちら / そちら / あちら / どちら)**：
  - 比 `这里/那里/哪里` 更礼貌的说法是“这边/那边/哪边”。
  - 常用于商务接待、指引方向，或者用来尊称对方（例如：`そちら` 可以指“您那边”或“您所在的团队/公司”）。

---

**本课总结：**

- 离自己近用 **こ** 系列：`ここ` (这里)
- 离对方近用 **そ** 系列：`そこ` (那里)
- 离大家都远用 **あ** 系列：`あそこ` (那里)
- 疑问词用 **ど** 系列：`どこ` (哪里)

记住这个 **“こ・そ・あ・ど”** 的规律，就能轻松掌握日语的指代体系了。

# 4. 存在句与位置表达

本课重点掌握如何区分“有生命”和“无生命”事物的存在表达，以及如何精确描述物体所在的具体方位。

### 1. 存在句：“あります” 和 “います”

日语中表示“有”或“在”时，必须根据对象的性质选择动词，这是最基础的区别点。

- **あります**：用于**无生命**的事物（如：书、植物、车辆）。

  - _例_：本（ほん）が **あります**。（有书。）
- **います**：用于**有生命**的事物（如：人、动物）。

  - _例_：猫（ねこ）が **います**。（有猫。）

### 2. 方位名词：上 / 下 / 前 / 後ろ...

要描述具体的空间位置，通常配合助词 `の` 使用，结构为 `名词 + の + 方位词`。

- **常用方位词**：

  - **上（うえ）** / **下（した）**：上面 / 下面
  - **前（まえ）** / **後ろ（うしろ）**：前面 / 后面
  - **中（なか）** / **外（そと）**：里面 / 外面
  - **隣（となり）**：旁边（通常指同类事物紧挨着）
- _注意_：“上”不仅表示垂直上方，接触表面的“在...桌子上”也用“上”表示。

### 3. 并列助词：名 と 名

用于连接两个名词，相当于中文的“和”。

- **用法**：`名词A と 名词B`
  - _例_：時計（とけい）**と** 眼鏡（めがね）。（表**和**眼镜。）

### 4. 全面否定：疑问词 + も + 动词(否定)

想要表达“什么都没有”、“谁都不在”时，使用这个句型。

- **结构**：疑问词（什么/谁/哪里）+ 助词 `も` + 否定形式。
- **例句**：
  - **何（なに）も** ありません。（**什么也**没有。）
  - **だれも** いません。（**谁也**不在。）

### 5. 确认助词：ね

放在句尾，表示征求对方的同意或确认事实，读升调。

- **语感**：相当于中文的“……吧？”或“……对不对？”
  - _例_：あそこ に 犬（いぬ）が います **ね**。（那边有只狗，是吧？）

### 6. 实用表达与礼貌用语

本课还介绍了一些丰富会话细节的表达：

- **思考时的填补词**：**ええと**。相当于中文的“嗯……”、“这个嘛……”，用于思考下一句怎么说时。
- **家庭称呼的敬语**：在提到别人的家庭成员时，为了表示礼貌，常在词头加 **ご**。

  - **ご家族（かぞく）**：您的家人。
  - **ご両親（りょうしん）**：您的父母。
  - **ご兄弟（きょうだい）**：您的兄弟姐妹。
- **生活状态**：**一人暮（ひとりぐ）らし**。意思是一个人生活、独居。

---

**本课总结：**

- 描述无生命事物（死物）的存在用 **あります**。
- 描述有生命事物（活物）的存在用 **います**。
- 描述位置关系用 **上 / 下 / 前 / 後ろ**。
- 连接两个名词用助词 **と**。
- 表达“什么都/谁都”没有，用助词 **も** 搭配否定形。
